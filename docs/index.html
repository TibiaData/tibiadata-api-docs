OVERVIEW

TibiaData API is a RESTful API providing information in JSON format containing information from Tibia's official homepage tibia.com, so you can build your own small tools.

The source is available on GitHub | Join our Discord server

Access the (modified) swagger.json for docs.tibiadata.com | Generic swagger document can be found in the GitHub release
